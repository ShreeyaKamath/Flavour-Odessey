# Changelog

All notable changes to Flavor Odyssey are tracked here.

## 1.0.0 - Portfolio MVP

### Added

- Joy Meadow MVP gameplay loop.
- Authentication foundation with guest mode.
- OpenAPI contract generation and typed frontend API usage.
- AI-ready deterministic provider architecture.
- Storybook UI, Lumi companion, weather/time, audio, animation, crafting, cinematic, and rendering foundations.
- Production readiness fallbacks for storage, audio, WebGL, assets, and offline mode.
- Portfolio showcase mode with architecture viewer, feature overview, screenshot plan, about page, credits, and release information.
- GitHub showcase documentation and community files.

### Known Limitations

- Final owned art and audio are not included.
- Only Joy Meadow is playable.
- AI uses deterministic fallback/mock behavior unless a future provider is configured.

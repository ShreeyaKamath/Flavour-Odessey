# Release Notes

## Version 1.0 Portfolio MVP

Flavor Odyssey Version 1.0 is a portfolio-ready MVP centered on Joy Meadow.

### Highlights

- Playable Joy Meadow vertical slice.
- Authentication foundation with guest mode.
- OpenAPI-generated frontend contracts.
- AI-ready deterministic dialogue, companion, recipe, memory, and narrative systems.
- Storybook UI transformation.
- Living weather/time and Lumi companion presentation.
- Magical crafting and cinematic foundations.
- Production readiness and browser fallback behavior.
- Portfolio showcase mode and release documentation.

### Validation

The release should pass:

- `npm run format`
- `npm run typecheck`
- `npm run lint`
- `npm test`
- `npm run build`

Backend tests are required when backend code changes.
